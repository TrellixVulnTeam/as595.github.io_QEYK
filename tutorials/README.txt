
Instructions for Exercises
---------------------------


 - The order of the exercises is:

1. PulsarClassifier.ipynb
2. GPMCarbonDioxide.ipynb
3. FRClassifier.ipynb

 - All the exercises use python jupyter notebooks. To launch a notebook from the
  command line:

 > jupyter notebook myfile.ipynb

 - For all exercises: first run through the notebook and try to understand what
 is happening at each step, then go back to the beginning of the notebook and
 work through the numbered exercises in order.
